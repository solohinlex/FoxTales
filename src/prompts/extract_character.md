# Извлечение и подготовка данных о персонажах

Твоя задача — извлечь данные о персонажах из текста и подготовить
компактный дайджест для литературного советника.
Не анализируй, не оценивай, не давай рекомендаций.

## Шаг 1 — Извлечение

Для каждого персонажа собери сырые данные:

**Внешность:**
- Только прямые описания — цитата + абзац
- Если не описано — пропустить

**Речь:**
- Все прямые цитаты с номером абзаца
- Манера речи — только если явно выражена (цитата)

**Действия:**
- Список по порядку — кратко + абзац

**Эмоции и состояние:**
- Только прямые указания — цитата + абзац
- Не интерпретировать

**Отношения:**
- С кем взаимодействует и как — цитата или описание + абзац

**Противоречия:**
- Если персонаж описан по-разному в разных местах
- Цитата 1 (абз. X) vs Цитата 2 (абз. Y)

## Шаг 2 — Фильтрация

Из сырых данных оставь только то, что важно для литературного анализа:

**Оставить:**
- Противоречия в описании персонажа
- Нетипичные действия или реакции — то что выбивается из общего тона
- Ключевые моменты развития персонажа
- Важные реплики — те что раскрывают характер или двигают сюжет
- Значимые отношения — конфликт, зависимость, перемена

**Убрать:**
- Проходные действия (вошёл, сел, посмотрел)
- Повторяющиеся эмоции без развития
- Описания внешности если они не несут смысловой нагрузки
- Реплики без контекста или значения

## Формат дайджеста

Никакого вводного текста. Только данные.

---
ПЕРСОНАЖИ: [имена через запятую]

[Имя персонажа]
Роль в тексте: [protagonist / supporting / antagonist / unclear]
Присутствие: [прямое / косвенное]

Ключевые моменты:
- [момент — цитата или описание (абз. X)]

Речь (значимые реплики):
- «[цитата]» (абз. X) — [почему значима: одним словом]

Отношения:
- [с кем]: [суть — одна фраза + абз. X]

Противоречия:
- [описание противоречия]
  «[цитата 1]» (абз. X) vs «[цитата 2]» (абз. Y)

Без имени:
- [как обозначен] — [что делает] (абз. X)
---

Если данных по разделу нет — раздел пропустить целиком.

## Пример дайджеста

---
ПЕРСОНАЖИ: Арен, Незнакомец, Мастер Орин (косвенно)

Арен
Роль в тексте: protagonist
Присутствие: прямое

Ключевые моменты:
- Прячется вместо того чтобы уйти — нарушает собственное правило (абз. 5)
- Впервые действует без совета Орина (абз. 8)

Речь (значимые реплики):
- «Я не боюсь темноты» (абз. 7) — противоречие (прячется в темноте абз. 5)

Отношения:
- Орин: зависимость ослабевает — «впервые без его слов» (абз. 8)
- Незнакомец: избегает, но наблюдает (абз. 5–6)

Противоречия:
- Видимость в темноте:
  «не видел ничего» (абз. 2) vs «заметил алый подол» (абз. 5)

Незнакомец
Роль в тексте: unclear
Присутствие: прямое

Ключевые моменты:
- Идёт не останавливаясь — не замечает Арена или игнорирует (абз. 6)

Без имени:
- «страж у ворот» — упомянут Ареном как ориентир, не появляется (абз. 3)
---
